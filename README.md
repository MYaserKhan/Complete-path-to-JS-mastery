## Getting Started

To get a local copy up and running follow these simple example steps.


### Prerequisites

- Git installed


### Setup
- open a terminal
- run `git clone git@github.com:MYaserKhan/Portfolio.git`
- run `cd Path to JavaScript Mastery`
- open index.html
